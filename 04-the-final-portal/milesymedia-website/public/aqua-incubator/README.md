# Aqua Incubator template — cover assets

Default cover imagery for the Aqua Incubator client-onboarding template
(chapter §15 / website-editor R002). The template references three paths
relative to `/`:

- `/aqua-incubator/cover-roots.jpg` — wide hero banner (forest / nature
  / dark-marble-with-gold textures).
- `/aqua-incubator/icon-incubator.png` — page icon chip (~64×64,
  landmark/badge art).
- `/aqua-incubator/card-cover.jpg` — 16:9 card-grid cover.

Drop final assets into this folder before publishing a Live client.
Operators can override per page via the editor properties panel.
Until then, the editor renders a graceful placeholder (browser shows a
broken-image marker; surrounding card chrome renders correctly).
