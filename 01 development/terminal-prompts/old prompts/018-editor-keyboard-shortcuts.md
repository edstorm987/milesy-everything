Cmd-K palette + per-block keyboard shortcuts. Operator productivity layer.

## Scope

**A** — Cmd-K (or Ctrl-K) opens command palette: fuzzy search over
"Insert <block>" / "Switch variant" / "Save" / "Publish" / "Toggle
preview" / "Undo" / "Redo".

**B** — Per-block shortcuts: `D` duplicate, `Del` delete, `[/]` move
up/down in tree, `Cmd-↑/↓` swap with sibling, `Esc` deselect.

**C** — Help footer "?" surfaces full shortcut list as a modal.

**D** — Smoke + chapter `04-editor-shortcuts.md` + MASTER row.
